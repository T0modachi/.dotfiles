# Keybindings

Window manager keybindings for Niri, driven by `home/niri.nix`.

## Navigation

| Key | Action |
|-----|--------|
| `Mod + H` / `Mod + Left` | Focus column left |
| `Mod + L` / `Mod + Right` | Focus column right |
| `Mod + J` / `Mod + Down` | Focus window down |
| `Mod + K` / `Mod + Up` | Focus window up |

## Move columns/windows

| Key | Action |
|-----|--------|
| `Mod + Shift + H` | Move column left |
| `Mod + Shift + L` | Move column right |
| `Mod + Shift + K` | Move window up |
| `Mod + Shift + J` | Move window down |

## Resize

| Key | Action |
|-----|--------|
| `Mod + Ctrl + H` | Decrease column width by 5% |
| `Mod + Ctrl + L` | Increase column width by 5% |
| `Mod + Ctrl + J` | Decrease window height by 5% |
| `Mod + Ctrl + K` | Increase window height by 5% |

## Workspaces (`w0`–`w9`)

| Key | Action |
|-----|--------|
| `Mod + 1`–`0` | Focus workspace `w0`–`w9` |
| `Mod + Shift + 1`–`0` | Move column to workspace `w0`–`w9` |

## Window management

| Key | Action |
|-----|--------|
| `Mod + Q` | Close window |
| `Mod + F` | Maximize column |
| `Mod + G` | Fullscreen window |
| `Mod + V` / `Mod + Shift + F` | Toggle window floating |
| `Mod + Shift + V` | Switch focus between floating and tiling |
| `Mod + C` | Center column |

## Apps and launcher

| Key | Action |
|-----|--------|
| `Mod + Return` | Open Ghostty terminal |
| `Mod + S` | Toggle Noctalia launcher |
| `Mod + Shift + B` | Open Zen Browser |
| `Mod + Shift + T` | Open Telegram |
| `Mod + Shift + D` | Open Discord |
| `Mod + Shift + A` | Open Pavucontrol |

## Screenshots

| Key | Action |
|-----|--------|
| `Mod + Shift + S` | Screenshot selected area to clipboard |
| `Mod + Ctrl + S` | Screenshot whole screen to clipboard |
| `Mod + Ctrl + E` | Edit clipboard image in Swappy |

## Media keys

| Key | Action |
|-----|--------|
| `XF86AudioRaiseVolume` | Raise volume 5% |
| `XF86AudioLowerVolume` | Lower volume 5% |
| `XF86AudioMute` | Toggle audio sink mute |
| `XF86AudioMicMute` | Toggle audio source mute |
| `Mod + W` | Mute microphone via Noctalia |
| `XF86MonBrightnessUp` | Increase brightness 10% |
| `XF86MonBrightnessDown` | Decrease brightness 10% |

## Mouse wheel

| Key | Action |
|-----|--------|
| `Mod + WheelScrollDown` | Focus column left |
| `Mod + WheelScrollUp` | Focus column right |
| `Mod + Ctrl + WheelScrollDown` | Focus workspace down |
| `Mod + Ctrl + WheelScrollUp` | Focus workspace up |

## Session / misc

| Key | Action |
|-----|--------|
| `Mod + Shift + E` / `Ctrl + Alt + Delete` | Quit Niri |
| `Mod + Escape` | Toggle keyboard shortcuts inhibit |
| `Mod + Shift + P` | Power off monitors |
| `Mod + O` | Toggle overview |
| `Mod + Shift + /` | Show hotkey overlay |
